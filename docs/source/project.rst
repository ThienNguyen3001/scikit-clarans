Project
=======

scikit-clarans is a small, focused library that provides a scikit-learn
compatible implementation of the CLARANS (Clustering Large Applications
based on RANdomized Search) algorithm. The documentation you are
reading contains usage guides, API reference and a gallery of
examples demonstrating common workflows.
